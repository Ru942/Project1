# Registration Form with Real-Time Validation

A responsive and interactive **Registration Form** built using **HTML, CSS, and JavaScript**. This project demonstrates real-time form validation, ensuring users enter valid information before submitting the form. The interface features a modern glassmorphism design with smooth animations, responsive layouts, and enhanced user experience.

## 🚀 Live Demo

**GitHub Pages:** https://ru942.github.io/Project1/

---

## 📌 Features

* 👤 Name field validation (cannot be empty)
* 📧 Email validation using a regular expression
* 🔒 Password validation (minimum 6 characters)
* ⚡ Real-time validation while typing
* 🚫 Submit button remains disabled until all fields are valid
* 👁️ Show/Hide password functionality
* 💪 Password strength indicator (Weak, Medium, Strong)
* ✅ Success message displayed after successful registration
* 🔄 Reset button to clear the form
* 🎨 Modern Glassmorphism UI with smooth animations
* 📱 Fully responsive design for desktop and mobile devices

## 📖 Validation Rules

### Name

* Must not be empty.
* Displays an error message if left blank.

### Email

* Must follow a valid email format.
* Displays an error message for invalid email addresses.

### Password

* Must contain at least **6 characters**.
* Displays an error message if the password is too short.
* Shows password strength based on length.

### Submit Button

* Disabled by default.
* Automatically enabled only when all input fields are valid.
